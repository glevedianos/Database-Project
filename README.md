# Recruitment Agency
This Code accompanies the project for "Database" Course, which took place in Electrical and Computer Engineering School at University of Patras in 2022

## Code File Descreption
The python files are separated depending on their usage.\ The first one creates the Class used for the Database, the second one creates the Database, the third one inserts data in the Database and the final one is a simple Graphic User Interface which accompanies the Database
